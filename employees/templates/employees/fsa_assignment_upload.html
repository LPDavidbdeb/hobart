CSV File Format

The CSV file must contain exactly two columns in this order:

  1. FSA Code: The 3-character code of the FSA (e.g., 'G1A').
  2. Employee Code: The unique code of the employee with the 'Manager' role.

The first row can optionally be a header row; it will be skipped automatically.
