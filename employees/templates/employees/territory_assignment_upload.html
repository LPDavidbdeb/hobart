CSV File Format

The CSV file must contain exactly two columns in this order:

  1. Territory Code: The unique code of the Territory (e.g., 'EAST').
  2. Manager Employee Code: The unique code of the employee with the 'Manager' role.

The first row can optionally be a header row; it will be skipped automatically.
